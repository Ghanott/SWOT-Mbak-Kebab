# Mbak Kebab Sleman Selatan Data Package

Paket data untuk analisis potensi bisnis Mbak Kebab di Sleman Selatan menggunakan SWOT Analysis.

## Cara Pakai Cepat

1. Pakai `landing_page_data.json` untuk frontend.
2. Pakai `zone_scoring.csv` untuk chart ranking zona.
3. Pakai `swot_evidence.csv` untuk membuat SWOT otomatis dengan Python.
4. Pakai `methods_code_snippets.py` untuk mengambil data OSM/Overpass asli.
5. Validasi manual data harga/menu Mbak Kebab sebelum final presentasi.

## Catatan
Data ini adalah paket baseline hasil riset publik. Beberapa data, terutama menu/harga dan koordinat POI, perlu divalidasi ulang sebelum digunakan sebagai data final akademik/bisnis.
